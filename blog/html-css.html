What are the best practices associated with using classes vs ids?

In this tech blog I will talk about 2 CSS attributes that can be used to style a web page and best practice associated with using these. The atributes are Class's and Id's.

When considering if Class or Id attribute should be used, the question to ask is: "is there more than one element that requires the same style? If the answer is yes, then Class attribute is best applied. Id attribute is used to apply styles to only one element.
