Explain the difference between accessing data from arrays and objects.

Objects and Arrays are a container for data.
Objects are used for storing items with their properties and arrays are used for storing itmes in order.

OBJECTS feature a Key:Value pair system. A 'key' is the name of the item in the container and is given a 'value'.
This is expressed as key:value;
Here is an example

    let car ={
    make: 'Toyota',
    model: 'Yaris',
    year: 2013 }
To access the data out of objects, you would express it like this:
car.make which will give you the make 'Toyota'
or car.model will result 'Yaris'

Unlike Objects, ARRAYS are a list that is in order. Arrays are ordered in square brackets.

    For example let cars = ['Toyota', 'Nissan', 'Suzuki', 'Mazda']
We get information out of arrays by using index position of items inside the array. Items are counted from Zero in an array. ie- toyota's index position is 0.
